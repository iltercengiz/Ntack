# Ntack

This repository contains several data structure and algorithm challenges written in C and Objective-C.

## Challenges

- Sum of pairs

Check if the array contains two elements whose sum equals to the given number.

- Stack

A very basic stack implementation that provides operations like push, pop, and peek.

- [Rotation](https://www.hackerrank.com/challenges/ctci-array-left-rotation)

Given an array of elements and a number of rotations, rotate the contents of the array to the left.
